A data visualization project that maps player movement from game telemetry logs onto an in-game map and helps analyze player behavior.

---

## 📌 Project Overview

This project processes `.nakama-0` game data (Parquet format) and visualizes player movement on a real game map using Python and Streamlit.

It helps in understanding:
- Player movement patterns
- Position tracking over time
- Game space navigation behavior

---

## ✨ Features

- 📂 Upload `.nakama-0` file
- 📊 Decode and process parquet data
- 🗺️ Overlay player positions on map image
- 📈 Plot movement trajectory (path tracking)
- 🔍 Debug coordinate statistics (min/max/mean)
- 📐 Proper map-data alignment

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- PyArrow
- Matplotlib
